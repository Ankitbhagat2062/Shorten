# Figma AI Prompts for Shorten - URL Shortener Project

## Page 1: Home Page
```
Design a modern and eye-catching homepage for a URL shortener called "Shorten". The page should include:
- A clean navigation navbar with logo, links (Home, Dashboard, Analytics, Profile, Login/Signup)
- Hero section with a compelling headline and subheadline
- A centered URL shortening form with two input fields: "Short URL Name" (custom alias) and "Original URL" with a prominent "Shorten" button
- An attractive registration banner explaining benefits: "Register to save your data - Never lose your shortened URLs when switching devices"
- Features section showcasing: No Registration Required, Analytics, Fast & Secure
- Modern footer with links and copyright
Use a vibrant color scheme with gradients (purple to blue), modern typography, and subtle animations. Make it fully responsive with glassmorphism effects on cards.
```

## Page 2: Dashboard Page
```
Design a sleek dashboard for a URL shortener web app called "Shorten". The dashboard should include:
- Sidebar navigation with icons (Dashboard, Analytics, Profile, Settings, Logout)
- Main content area showing:
  - Stats cards row: Total URLs Created, Total Views/Clicks, Active Links
  - Recent URLs table with columns: Short Name, Original URL, Clicks, Created Date, Actions
  - Each row should have "Edit" and "Delete" action buttons
- A "Create New Short URL" quick action button
- Search and filter options for URLs
- Toast notifications area
Use a dark theme with accent colors, card-based layout with subtle shadows, data visualization elements. Modern UI with rounded corners and hover effects.
```

## Page 3: Analytics Page (Main Analytics)
```
Design a comprehensive analytics dashboard for a URL shortener called "Shorten". Include:
- Overview stats cards: Total Clicks, Return Users, Real Users, Peak Visit Time
- Interactive world map showing user locations with heat indicators
- Device breakdown chart (Mobile, Desktop, Tablet) with percentages
- Browser usage statistics
- Hourly/Daily/Weekly click activity graph (line chart)
- Top performing URLs list
- Time-based analytics showing "At what time users visit most" as a bar chart
Use a professional color palette, clean data visualizations, filter options by date range. Modern dashboard aesthetic with cards, charts, and metric highlights.
```

## Page 4: URL Analytics Page (Individual URL)
```
Design a detailed analytics page for a single shortened URL in "Shorten". Include:
- URL info card showing: Short URL, Original URL, Created Date, Total Clicks
- Individual stats: Total Clicks, Unique Visitors, Return Visitors
- Geographic breakdown with country flags and click counts
- Device information: OS types (iOS, Android, Windows, Mac, Linux), Browser types
- Referrer sources (where users came from)
- Timeline view showing click activity over time
- Conversion tracking if applicable
Use a cohesive design with the main analytics page, include export functionality button. Professional data presentation with tables and visual charts.
```

## Page 5: Short URL Name Page (Redirect Page)
```
Design a modern redirect/landing page for a shortened URL in "Shorten". Include:
- Centered loading animation during redirect
- URL preview card showing: Short URL name, destination URL (masked initially), safety indicator
- "Taking you to: [original URL]" message
- Skip/Continue button
- Footer with "Powered by Shorten" branding
- Optional: Ad or promotional space
Use a clean, minimal design with smooth transition animations. Include privacy indicator and security badge. Make it mobile-friendly with quick loading animation.
```

## Page 6: SignUp/Login Page
```
Design a modern authentication page for "Shorten" with both signup and login options. Include:
- Split layout: Left side with branded illustration/pattern, Right side with form
- Tab toggle between "Sign In" and "Sign Up"
- Login form: Email/Username field, "Continue" button
- Signup form: Name, Email fields
- Social login options displayed prominently: "Continue with Google" (with Google icon), "Continue with GitHub" (with GitHub icon)
- "Continue with Email OTP" option - no password required
- Privacy policy and terms links
- Registration benefits banner: "Register to save your data across devices"
Use a modern, clean aesthetic with gradient accents. Make social buttons distinctive with brand colors. Include form validation states and error messages. Fully responsive design.
```

---

## Additional Tips for Figma AI:
1. **For each prompt**: Copy and paste directly into Figma AI or similar AI design tools
2. **Style consistency**: Add "Use consistent design system with [color hex codes]" if you have brand colors
3. **Mobile responsiveness**: Add "Ensure mobile-first responsive design" to any prompt
4. **Accessibility**: Add "Include proper contrast ratios and accessible design patterns"
